## degiskenler
```python
# str() parantez icine yazilan degiskeni metinsel ifadeye cevirir
plaka = 65
plaka = str(plaka) # metinsel ifadeye donusturdu.
# plaka = "65"
print(plaka + "1")
```

```python
# int() parantez icine yazilan degiskeni sayisal ifadeye cevirir
yas = "15"
yas = int(yas)
print(yas + 1)
```
